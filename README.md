# 🏨 Hotel Landing Page

A responsive hotel landing page built from scratch using pure HTML and CSS — no frameworks, no templates. Developed as a portfolio project to demonstrate front-end web development skills.

## 📸 Preview

> A modern, elegant hotel website featuring smooth scroll animations, a video section, room listings, and a footer with contact details.

---

## ✨ Features

- **Hero section** with full-screen background image and entrance animations
- **Welcome section** with image and embedded video
- **Accommodations** — room cards with photos and pricing
- **Family Suite** showcase with multiple room photos and captions
- **Promos & Offers** section
- **Footer** with contact info, office hours, and social links
- **Scroll animations** powered by [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)
- **Icons** via [Font Awesome 7](https://fontawesome.com/)
- **Google Fonts** — Merriweather serif typeface

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Page structure and semantics |
| CSS3 | Custom styling, backgrounds, layout |
| Tailwind CSS | Utility-first responsive layout |
| AOS.js | Scroll-triggered animations |
| Font Awesome | Icons |
| Google Fonts | Typography |

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/cpplover52/hotel-website.git
   ```

2. Open `index.html` in your browser — no build step required.

> **Note:** If you want to modify Tailwind classes, you'll need the Tailwind CLI to recompile `output.css`.
